# Student Management System

A console-based Student Management System developed in C++ using file handling.

## Features

- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student
- Persistent storage using text files

## Technologies Used

- C++
- File Handling
- Object-Oriented Programming

## How to Run

```bash
g++ student.cpp -o student
./student
```

## Project Structure

- student.cpp - Source code
- students.txt - Stores student records
